# 🌿 Plant Disease Detector

## 🚀 Overview

This project uses Machine Learning / Deep Learning to detect plant diseases from leaf images.
It helps in early diagnosis, improving crop health and reducing losses for farmers.

---

## 🧠 Features

* 📸 Image-based disease detection
* ⚡ Fast predictions
* 🌐 Simple UI using Streamlit
* 🔄 (Optional) Fallback system using manual inputs

---

## 🛠 Tech Stack

* Python
* TensorFlow / Keras / Scikit-learn
* OpenCV
* Streamlit

## ⚡ How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/plant-disease-detector.git
cd plant-disease-detector
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the app:

```bash
streamlit run app.py
```
